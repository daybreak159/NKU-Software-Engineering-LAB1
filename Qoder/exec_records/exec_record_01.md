### 步骤 2-4：创建三个主题文件

**notes1.txt（任务规划）**
```
任务规划能力评测要点

任务规划是 AI Agent 的核心能力之一，要求 Agent 能够：
将复杂任务分解为可执行的原子步骤
识别步骤之间的依赖关系和执行顺序
根据中间结果动态调整后续计划
在资源受限时合理分配优先级

评测时需关注步骤完整性、顺序合理性和异常处理能力。
```

**notes2.txt（工具调用）**
```
工具调用能力评测要点

工具调用能力体现 Agent 与外部系统交互的水平，包括：
正确识别需要使用工具的场景
选择合适的工具并构造正确参数
解析工具返回结果并融入后续推理
处理工具调用失败时的降级策略

评测重点在于参数准确性、错误处理和调用效率。
```

**notes3.txt（记忆与上下文管理）**
```
记忆与上下文管理能力评测要点

记忆与上下文管理决定 Agent 能否在长对话中保持一致性：
短期记忆用于维护当前会话的关键信息
长期记忆支持跨会话的知识积累与检索
上下文窗口管理需要在信息保留与成本间平衡
遗忘机制避免无关信息干扰当前任务

评测时需验证跨轮信息保持、冲突检测和上下文压缩能力。
```
