# Qoder CLI 评测记录

本目录包含 Qoder CLI 的评测执行记录和测试数据。

## 内容

- **exec_records/** — Q1-Q9 共 9 个 md 文件，每题包含用户指令和执行过程
- **data/** — 测试数据（notes1-3.txt 三个主题文件）
- **code_eval/** — Bug 修复任务文件（scores.csv、score_report.py、repair_report.md）
- **Qoder02~08.png** — 7 张执行过程截图
- **eval_design.md** — 评测设计说明
- **summary.md / result.csv / stats.py / stats.txt** — 各任务产生的统计结果
